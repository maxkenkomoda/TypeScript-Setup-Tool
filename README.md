# Tools to init TypeScript

## How to run

1. clone this tool

```
git@github.com:maxkenkomoda/TypeScript-Setup-Tool.git
```

2. move directory

```
cd TypeScript-Setup-Tool
```

3. input name of libraries that you require for project in libs.txt.

4. run shell script

```
sh init.sh
```

5. change your project name

- package.json
- directory name
